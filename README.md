# Life Expectancy vs GDP (2000–2015)

## English

**Goal**  
Explore the relationship between GDP and life expectancy across 6 countries.

**Data**  
- `all_data.csv`  
- Columns: Country, Year, Life expectancy at birth (years), GDP (current US$)

**Tools**  
- pandas  
- seaborn  
- matplotlib  

**Outputs**  
- Time series plots  
- Scatter plot (GDP vs Life Expectancy)  
- Distribution plot  
- Conclusions (EN + HU)

**How to run**  
Open `life_expectancy_gdp.ipynb` in Jupyter and run all cells.
